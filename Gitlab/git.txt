EXPERIMENT 1: Setting Up and Basic Git Commands

    Commands used:

    git init

    git add .

    git commit -m "initial commit"

EXPERIMENT 2: Creating and Managing Branches

    Commands used:

    git checkout -b main

    git checkout master

    git merge main

EXPERIMENT 3: Stashing Changes

    Commands used:

    git stash save "stashed"

    git checkout main

    git stash apply

EXPERIMENT 4: Working with Remote Repositories (Clone & Remote Setup)

    Commands used:

    git remote add origin https://github.com/Khushi256/new.git

    git fetch origin

EXPERIMENT 5: Fetching and Rebasing from Remote Repository

    Commands used:

    git fetch origin

    git rebase origin/main

EXPERIMENT 6: Merging with Custom Commit Message

    Commands used:

    git checkout master

    git merge main


EXPERIMENT 7: Git Tags and Releases

    Commands used:

    git tag v1.0

    git log --oneline

EXPERIMENT 8: Cherry-Picking Commits

    Commands used:

    git log --oneline

    git cherry-pick aaaa677

    git cherry-pick --skip

EXPERIMENT 9: Viewing Details of a Specific Commit

    Commands used:

    git log -n 1 d54bbdf

EXPERIMENT 10: Listing Commits by Author and Date Range

    Commands used:

    git log --author="JohnDoe" --since="2023-01-01" --until="2023-12-31"

EXPERIMENT 11: Displaying Recent Commit History

    Commands used:

    git log --oneline

    git log -n 5

EXPERIMENT 12: Undoing a Commit (Revert Operation)

    Commands used:

    git revert 6818b20

    git commit (opened automatically after revert)
