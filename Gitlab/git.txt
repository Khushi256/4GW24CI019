
## **1. Setting Up and Basic Commands**

```bash
git init
```

```bash
touch file.txt
```

```bash
git add file.txt
```

```bash
git commit -m "Initial commit with file.txt"
```

---

## **2. Creating and Managing Branches**

```bash
git branch feature-branch
```

```bash
git checkout master
```

```bash
git merge feature-branch
```

---

## **3. Creating and Managing Branches (Stashing Changes)**

```bash
git stash
```

```bash
git checkout feature-branch
```

```bash
git stash apply
```

```bash

```

ðŸ“Œ *Replace `commit1` and `commit5` with actual commit IDs.*

## **10. Analysing and Changing Git History (Commits by Author in Date Range)**

```bash

```

## **11. Analysing and Changing Git History (Last Five Commits)**

